# Results Plan

The evaluation should demonstrate that multimodal integration improves over single-modality baselines.

Suggested comparisons:

- EHR-only model.
- Imaging-only model.
- Wearable-only temporal model.
- Text-only model.
- Full multimodal attention-fusion model.

Suggested metrics:

- Accuracy.
- Precision.
- Recall.
- F1-score.
- AUC.
- Calibration error for probability outputs.

The strongest result would show that the full model improves predictive performance while staying robust when one or more modalities are unavailable.
