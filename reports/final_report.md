# Final Report

## Summary

This project combines heterogeneous health data into one predictive framework for earlier and more reliable disease-risk detection. The main strengths are multimodal learning, robustness to missing inputs, and explainability for clinical review.

## Expected Evaluation

The full multimodal model should be compared against single-modality baselines such as EHR-only, image-only, and wearable-only models. Suggested metrics include accuracy, precision, recall, F1-score, AUC, and calibration.

## Clinical Positioning

The system should be presented as a clinical decision-support tool. Its purpose is to flag elevated risk, reveal important contributing signals, and help clinicians prioritize high-risk patients sooner.
